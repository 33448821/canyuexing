# 点餐程序

本项目是一个点餐程序，包括前后端分离的网页版和微信小程序版。

## 功能特点

- 用户注册登录
- 查看菜单并点餐
- 下单并支付
- 查看订单状态
- 管理员权限：管理菜单、订单管理

## 技术栈

- 前端：Vue.js、微信小程序框架
- 后端：Node.js、Express框架
- 数据库：MySQL

## 如何运行

### 前端

```bash
# 进入前端目录
cd frontend

# 安装依赖
npm install

# 运行开发服务器
npm run serve
# 进入后端目录
cd backend

# 安装依赖
npm install

# 运行服务器
npm start

```
